Servidor Python com banco de dados ORM com SQLAlchemy 
e api Rest utilizando Flask

Requisitos:
- Python 3.8 
- Postgres 9.5
- virtualenv

python -m venv .venv
pip install pipenv

Pacotes para instalação estão tanto no Pipfile quanto no
requirements.txt

Para Executar basta acessar a pasta do server,
e digitar:
> python run.py
